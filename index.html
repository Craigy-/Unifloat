Unifloat — simply creating popup elements

Unifloat is a universal jQuery plugin which allows you to create various types of popup elements. For example: dropdown menu, tooltips with or without moving after the mouse cursor, or various popup windows with any content.

Unifloat developed like a little framework which give possibility to make your own code using plugin's methods and capabilities.
