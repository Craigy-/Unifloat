Unifloat - simply creating floating elements

Unifloat is a universal jQuery plugin which allows you to create various types of floating elements. For example: dropdown menu, tooltips with or without moving, or something else.

Unifloat designed like a little framework which give possibility to make your own code using plugin's methods and capabilities.
